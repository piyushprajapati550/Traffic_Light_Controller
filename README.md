Traffic Light Controller Top Module for FPGA
This top-level Verilog module integrates a 100 MHz to 1 Hz clock divider with the Traffic Light Controller FSM, enabling real-time traffic signal operation on an FPGA. It connects the onboard clock and reset inputs to the controller and drives four 3-bit traffic light outputs (M1, S, MT, M2). Designed for FPGA deployment without requiring a testbench.
